HTML Tags

The common HTML tags used to create and structure HTML pages.

Tag Name Description
<!DOCTYPE> Document Type Provides information to the browser about what document type to expect.
<head> Head Container for metadata. Loads character sets, styles, scripts and other meta information. Not displayed on the webpage.
<body> Body Defines the document's body, which contains all the contents of an HTML document.
<header> Header The header of a document. Webpage title, logos and navigation bars are displayed on the header.
<main> Main The main content of a document, should contain content that is not repeated across webpages.
<footer> Footer The footer of a document, typically contains contact details, copyright and authorship.
<section> Section Defines a section in a document.
<aside> Aside Defines some content aside from the content it is placed in, should be indirectly related to the surround content, such as a sidebar.
<article> Article Specifies self-contained, independent content. Supposed to make sense on its own and should be possible to distribute it independently from the rest of the site.
<div> div Defines a division or section in an HTML document. #div as a CSS selector
<class> class Points to a class name in a style sheet. Can also be used by JavaScript to access and manipulate elements with that specific class name.
<img /> Image Displays an image on a webpage.

<img src="source" alt="alternate text if image cannot be displayed" />
<video> Video Displays a video that can be played.

<video width="" height="" controls>
<source src="source" type="type">
</video>

HTML Tables

Tag Name Description
<table> Table Creates an HTML table, consisting of table rows, table headers and table data.
<tr> Table Row Creates a table row.
<th> Table Heading Creates a table heading.
<td> Table Data Creates a standard data cell in an HTML table.
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

CSS Colours

Colours can be specified using predefined colour names (e.g. red, blue, yellow, orange etc..) but aren't easily customisable nor are they the most precise. Below are other ways to declare colours that can be more precisely defined:

Tag Name Description
rgb(red, green, blue) Red, Green, Blue Specifies the red, green and blue light sources, defines the intensity of the colour between 0 and 255 for each.
rgba(red, green, blue, alpha) Red, Green, Blue, Alpha Extension of RGB colour with an alpha channel - specifies the opacity of the colour. 0.0 (full transparency) and 1.0 (opaque).
hsl(hue, saturation, lightness) Hue, Saturation, Lightness Specifies a colour using hue, saturation and lightness.

Hue: The degree on the colour wheel from 0 to 360. 0 = red. 120 = green. 240 = blue.
Saturation: The percentage value of grey to colour. 0% = a shade of grey. 100% = full colour.
Lightness: The percentage value of light to the colour. 0% = no light (black). 100% = full lightness (white).
##rrggbb Hexadecimal colour Specifies a colour using hexadecimal values, where rr, gg and bb are hex values between 00 and ff (same as decimal 0 to 255).
